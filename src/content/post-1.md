# Hackathon de Interledger en CDMX

El desafío consistió en crear una solución innovadora para transacciones interbancarias. Trabajar en equipo bajo presión, aprender nuevas tecnologías sobre la marcha, como el uso de una API, y presentar el proyecto ante jueces fue una experiencia que valió la pena por todo lo que conllevó y, sobre todo, por la experiencia adquirida.

## La Experiencia

Durante 48 horas intensas, nuestro equipo se enfocó en desarrollar **Open Wallet**, una plataforma que utiliza la API de Interledger para facilitar pagos interbancarios de manera segura y eficiente.

## Aprendizajes Clave

1. **Trabajo bajo presión**: Aprendi que el tiempo es clave para entregar un proyecto de este tipo.
2. **Nuevas tecnologías**: La API de Interledger dio problemas en algunos momentos.
3. **Presentación**: Comunicar ideas en inglés fue algo para lo que no estábamos preparados.

Esta experiencia nos demostró que estamos preparados como estudiantes para afrontar retos de este calibre, ya que participamos junto a profesionales y obtuvimos un buen resultado. Si bien tuvimos problemas técnicos que nos afectaron de manera significativa, esta experiencia nos sirve para mejorar en el futuro.
