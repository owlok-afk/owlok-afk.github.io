# Hackathon de Interledger en CDMX

El desafío consistió en crear una solución innovadora para transacciones interbancarias. Trabajar en equipo bajo presión, aprender nuevas tecnologías sobre la marcha y presentar ante jueces fue una experiencia que valió la pena por haber aprendido de una manera tan forzada.

## La Experiencia

Durante 48 horas intensas, nuestro equipo se enfocó en desarrollar **Open Wallet**, una plataforma que utiliza la API de Interledger para facilitar pagos interbancarios de manera segura y eficiente.

## Aprendizajes Clave

1. **Trabajo bajo presión**: Aprendimos a gestionar el tiempo y priorizar tareas críticas
2. **Nuevas tecnologías**: La API de Interledger nos presentó conceptos innovadores en pagos
3. **Presentación**: Comunicar ideas complejas de forma clara fue fundamental

Esta experiencia nos demostró que el aprendizaje acelerado en un ambiente de competencia puede ser tremendamente productivo.
